# wage-inequality-econometric-analysis
Econometric analysis of wage inequality using NLSY survey data with regression modeling in Python/R.
# Wage Inequality Econometric Analysis (NLSY Data)

## Overview
This project investigates determinants of wage inequality using data from the National Longitudinal Survey of Youth (NLSY). The analysis applies econometric regression techniques to examine how education, ability scores, and demographic factors influence wage outcomes.

## Dataset
The analysis uses NLSY survey data, a large longitudinal dataset tracking labor market outcomes of individuals over time.

Due to dataset licensing restrictions, the full dataset is not included in this repository.

## Tools Used
Python / R  
Pandas / Statsmodels / R regression tools  
Jupyter Notebook / RStudio

## Methodology
- Data cleaning and preprocessing
- Exploratory data analysis
- Econometric regression modeling
- Interpretation of statistical significance

## Key Findings
Education level and cognitive ability (AFQT scores) show strong positive relationships with wage outcomes. Gender differences in wages remain significant even after controlling for education and experience.
